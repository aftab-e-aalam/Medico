# 🏥 Medical Appointment System

A dynamic web application built using **Flask** that allows users to browse hospitals, view doctors, and manage authentication (login/register).

---

## 🚀 Features

* 🔐 User Authentication (Login & Register)
* 🏥 Hospital Listing (Dynamic from CSV/Database)
* 👨‍⚕️ Doctor Listing with Details
* ⭐ Rating-based Sorting
* 🎨 Responsive UI (Bootstrap + Custom CSS)
* 📊 Data-driven pages using Flask + Pandas

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Database:** SQLite (SQLAlchemy)
* **Frontend:** HTML, CSS, Bootstrap
* **Data Handling:** Pandas
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

---

### 2️⃣ Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate   # Windows
```

---

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

### 4️⃣ Run the App

```
python app.py
```
